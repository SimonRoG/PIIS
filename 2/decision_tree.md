# Двійкове дерево рішень для діагностики ВІЛ-інфекції

## Опис

Це дерево рішень представляє логіку діагностики ВІЛ-інфекції на основі симптомів та аналізів пацієнта. Дерево використовує ієрархічний підхід для класифікації ймовірності ВІЛ-інфекції на чотири рівні: дуже висока, висока, середня та низька.

## Структура дерева

## Детальна структура дерева рішень

### Рівень 1: Перевірка на дуже високу ймовірність

**Вузол 1.1**: Чи є у пацієнта >= 15 симптомів?
- **ТАК** → **ДУЖЕ ВИСОКА ЙМОВІРНІСТЬ ВІЛ**
- **НІ** → Перехід до Вузла 1.2

**Вузол 1.2**: Чи є всі з наступного:
- Симптоми мононуклеозу (ангіна + запалення лімфовузлів)?
- Аналізи мононуклеозу (підвищені лейкоцити АБО лімфоцити АБО атипові клітини)?
- Симптоми серозного менінгіту (нудота + блювання)?
- Симптоми енцефаліту (нудота + блювання)?
- Симптоми езофагіту (біль за грудиною + порушення ковтання)?

- **ТАК** → **ДУЖЕ ВИСОКА ЙМОВІРНІСТЬ ВІЛ**
- **НІ** → Перехід до Рівня 2

### Рівень 2: Перевірка на високу ймовірність

**Вузол 2.1**: Чи є 10 <= кількість симптомів < 15?
- **ТАК** → **ВИСОКА ЙМОВІРНІСТЬ ВІЛ**
- **НІ** → Перехід до Вузла 2.2

**Вузол 2.2**: Чи є хоча б одне з наступного:
- Мононуклеоз + аналізи мононуклеозу?
- Симптоми серозного менінгіту?
- Симптоми енцефаліту?
- Симптоми езофагіту?

- **ТАК** → **ВИСОКА ЙМОВІРНІСТЬ ВІЛ**
- **НІ** → Перехід до Рівня 3

### Рівень 3: Перевірка на середню ймовірність

**Вузол 3.1**: Чи є 6 <= кількість симптомів < 10?
- **ТАК** → **СЕРЕДНЯ ЙМОВІРНІСТЬ ВІЛ**
- **НІ** → Перехід до Вузла 3.2

**Вузол 3.2**: Чи є всі з наступних симптомів:
- Порушення сну
- Тяжкість у підреберях
- Ниючі болі у підреберях
- Понос
- Нудота
- Блювання

- **ТАК** → **СЕРЕДНЯ ЙМОВІРНІСТЬ ВІЛ**
- **НІ** → Перехід до Рівня 4

### Рівень 4: Низька ймовірність

**Вузол 4.1**: Чи є кількість симптомів < 6?
- **ТАК** → **НИЗЬКА ЙМОВІРНІСТЬ ВІЛ**
- **НІ** → Перехід до Вузла 4.2

**Вузол 4.2**: Чи є всі базові симптоми:
- Підвищена температура
- Ангіна
- Слабкість
- Головний біль

- **ТАК** → **НИЗЬКА ЙМОВІРНІСТЬ ВІЛ**
- **НІ** → **НИЗЬКА ЙМОВІРНІСТЬ ВІЛ** (за замовчуванням)

## Опис симптомів та захворювань

## Приклад проходження дерева

### Пацієнт: людина1
**Симптоми**:
- Підвищена температура
- Ангіна
- Запалення лімфовузлів
- Понос
- Нудота
- Блювання
- Біль за грудиною
- Головний біль
- Розбитість

**Аналізи**:
- Підвищений рівень лейкоцитів

**Проходження дерева**:

1. **Вузол 1.1**: Кількість симптомів = 9 < 15 → НІ
2. **Вузол 1.2**: 
   - Мононуклеоз: ангіна + запалення лімфовузлів → ТАК
   - Аналізи: підвищений рівень лейкоцитів → ТАК
   - Менінгіт: нудота + блювання → ТАК
   - Енцефаліт: нудота + блювання → ТАК
   - Езофагіт: біль за грудиною, але немає порушення ковтання → НІ
   - **Результат**: НІ (не всі умови виконані)

3. **Вузол 2.1**: 9 < 10 → НІ
4. **Вузол 2.2**: 
   - Мононуклеоз + аналізи → ТАК
   - **Результат**: ТАК → **ВИСОКА ЙМОВІРНІСТЬ ВІЛ**

**Висновок**: Високий ризик ВІЛ-інфекції

## Алгоритм використання дерева

1. Зібрати всі симптоми пацієнта
2. Зібрати всі аналізи пацієнта
3. Підрахувати загальну кількість симптомів
4. Перевірити наявність специфічних комбінацій симптомів для кожного захворювання
5. Пройти дерево рішень зверху вниз
6. Повернути перший підходящий рівень ймовірності

## Примітки

- Дерево використовує пріоритетну систему: спочатку перевіряються більш серйозні випадки
- Деякі симптоми можуть належати до кількох захворювань (наприклад, нудота та блювання є ознаками як менінгіту, так і енцефаліту)
- Система вимагає точної комбінації симптомів для діагностики специфічних захворювань
- Кількісний підрахунок симптомів використовується як додатковий критерій оцінки
